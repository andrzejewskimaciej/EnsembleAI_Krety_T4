# ECG Digitization Pipeline

Hackathon solution for Task 4 - automated digitization of paper ECG images into 12-lead 1D signals.

## Problem

Given an image of a paper ECG, output 12 standard leads as 1D digital signals with correct amplitude (mV) and time calibration.

The dataset contains 3000 training images with ground truth and 500 test images across three difficulty levels:
- **Easy** - high-resolution digital scans
- **Medium** - smartphone photos with noise and slight rotation
- **Hard** - crumpled paper, photocopies, handwritten annotations
